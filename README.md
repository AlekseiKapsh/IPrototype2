Интерфейс обязывает все классы иметь метод Clone, чтобы их можно было копировать единообразно
Классы хранят данные: простые (имя, цена) и сложные (список функций). Список это ссылка на объект в памяти
Метод Clone делает две вещи: автоматически копирует простые данные, но вручную создает новую копию списка функций. Это нужно, чтобы у оригинала и клона были разные списки в памяти
В программе создается один план, затем он клонируется. В копию добавляется новая функция
Вывод показывает, что новая функция появилась только у копии, а оригинал не изменился. Это доказывает, что объекты полностью независимы друг от друга
